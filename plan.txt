Основные сущности:

+ Вопрос.
    атрибуты:
    + автор (пользователь)
    + задан (data)
    + просмотров (number)
    + имеет набор тегов
    + коротка ссылка
    + ответ имеет рейтинг (+/-1)
    + количество просмотров (учет ведётся на основании просмотров зарег. пользователей)
    + вопрос может считаться отвеченным (указываем ответ)
    + любой вопрос можно добавить в избранные
    + если вопрос кто-то уже добавил в избранное, то показываем это рядом с вопросом
    + если вопрос отредактирован, то показываем последнюю дату редактирования

+ Ответы к вопросу.
    + автор (пользователь)
    + отвечено (дата)
    + короткая ссылка
    + ответ может быть ответам на вопрос (точнее лучшим ответом. Дата отметки как лучший - тоже показывается)
    + если ответ отредактирован, то показываем последнюю дату редактирования

+ Комментарии:
    + автор (пользователь)
    + дата
    + к чему относится
    + если автор редактировал, то сохраняем сколько раз это было и показываем
    + после редактирования комментария, мы начинаем вести логирование, которое будет сохранять каждую версию
      редактирования. Появляется рядом с комментом ссылка, которая ведет на просмотр истории редактирований.

+ Создание нового вопроса:

+ Пользователь.
    + репутация

Другие возможности:
    Вопросы и ответы можно комментировать отдельно (просто списком)
    Автор комментария может его редактировать или удлалить
    Автор вопроса/ответа не имеет права голосовать за свой вопрос/ответ
    Список ответ сортируем по след. критериям:
        + по кол-ву голосов (desc)
        + по дате (desc)

Главное меню.
    Поиск по тегам
    Поиск по пользователям
